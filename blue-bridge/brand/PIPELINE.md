## ANTI-PATTERNS

| Ban | Prohibición | Por qué |
|-----|-------------|----------|
| **The Globe Ban** | Globo terráqueo, mapa con puntos conectados | Cada BPO lo usa. Invisible. |
| **The Handshake Ban** | Fotos de stock, equipos sonrientes, reuniones de whiteboard | Wallpaper visual. No dice nada. |
| **The Warm Color Ban** | Coral, amber, naranja, verde (excepto cyan-green de paleta), rojo | Destruye la identidad de precisión fría. |
| **The Three Cards Ban** | 3 elementos simétricos idénticos | Default de template. Sin criterio. |
| **The Floating UI Ban** | Cards de UI flotando sin contenedor | Estética de Dribbble, no de firma de estrategia. |
| **The Generic Copy Ban** | Seamless, end-to-end, world-class, cutting-edge | El C-suite europeo lo ha leído 1000 veces. |
| **The Blue Pollution Ban** | Cualquier azul que no sea #091FF9 o #050D6B | La marca es azul eléctrico específico — no un azul cualquiera. |

## WORKFLOW

1. **Load** — leer DESIGN.md Sección 10 (lecciones de producción) antes de generar
2. **Brief** — responder Q1–Q5 de Fase 1 (marca vs. sector, tipo, formato, tema, copy)
3. **Research** — si Q4 = investigar, web search con fuentes verificables
4. **Brand Read** — verificar paleta activa, tono, audiencia contra DESIGN.md
5. **2 Direcciones** — proponer exactamente 2 opciones de la Visual Concept Library. Esperar selección explícita
6. **Copy** — HEADLINE: Space Grotesk 700 blanco max 8 palabras · LABEL: Inter 600 cyan-green uppercase · BODY: Inter 400 blanco max 3 líneas · CTA: Inter 600 cyan-green
7. **Approval Gate** — mostrar brief completo. Esperar confirmación explícita antes de generar
8. **Generación** — usar generar-imagen-recraft con template de prompt del concepto seleccionado
9. **Learning Log** — preguntar si el usuario quiere registrar lección en DESIGN.md Sección 10

## ITERATION PROTOCOL

| El usuario dice | El pipeline hace |
|-----------------|------------------|
| más formal | FORMALITY +1, copy más contenido, menos adjetivos |
| más limpio | Menos elementos secundarios, más espacio negativo |
| otro concepto | Volver a Fase 3, proponer 2 conceptos nuevos de la Visual Concept Library |
| cambia el copy | Solo copy nuevo — mantiene visual y layout |
| más datos | Añade dato o fuente al body sin cambiar composición |
| en español | Traduce copy — mantiene tipografía, paleta, layout |
| perfecto, genera | Confirmar prompt y proceder a generar-imagen-recraft |
